1. 弹性盒子(flexbox)

    2. 使用 d-flex 类创建一个弹性盒子容器，并设置三个弹性子元素：

    Flex item 1
    Flex item 2
    Flex item 3

    创建显示在同一行上的弹性盒子容器可以使用 d-inline-flex 类:

    Flex item 1
    Flex item 2

  2. 水平方向

    3. 使用 .flex-row 类设置弹性子元素水平显示：

    Flex item 1
    Flex item 2
    Flex item 3

    Flex item 1

  3. 垂直方向

    4. .flex-column 类用于设置弹性子元素垂直方向显示：

    Flex item 3

    6. .flex-fill 类强制设置各个弹性子元素的宽度是一样的:

    Flex item 1
    Flex item 2
    Flex item 3

    Flex item 1

  6. 扩展

    7. .flex-grow-1 用于设置子元素使用剩下的空间：

    Flex item 1

  7. 排序

    8. .order 类可以设置弹性子元素的排序，从 .order-1 到 .order-12，数字越低权重越高:
