五行学说最早在道家学说中出现。它强调整体概念，描绘了事物的结构关系和运动形式。

五行相生：金生水，水生木，木生火，火生土，土生金

  • 木生火：因为火以木料作燃料的材料，木烧尽，则火会自动消灭
  • 火生土：因为火燃烧物体后，物体化为灰烬而灰烬便是土
  • 土生金：因为金蕴藏于泥土石块之中，经冶炼后才提取黄金
  • 金生水：因为金若被烈火燃烧，便溶为液体，液体属水；水要依靠铁器来开导疏通
  • 水生木：因为水灌溉树木，树木便能欣欣向荣

五行相克：金克木，木克土，土克水，水克火，火克金

  • 水克火：因为火遇水便熄灭
  • 火克金：因为烈火能溶解金属
  • 金克木：因为金属铸造的切割工具可锯毁树木
  • 木克土：因为树根苗的力量强大，能突破土的障碍
  • 土克水：因为土能防水
